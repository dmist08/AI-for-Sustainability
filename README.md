# AI for Sustainability — SRIP 2026 Project

## Project Overview
This repository contains the complete implementation for the SRIP 2026 Earth Observation assignment — an AI-based audit of the Delhi Airshed for land-use classification using **Sentinel-2 satellite imagery**, **ESA WorldCover labels**, and a **ResNet-18 CNN** (11.2M parameters, ImageNet pretrained).

📄 **Full report with analysis:** [`report.pdf`](report.pdf)

## Running the Pipeline (Google Colab)

1. **Open** `notebooks/delhi_airshed_pipeline.ipynb` in Google Colab
2. **Set runtime** to **GPU → T4** (Runtime → Change runtime type)
3. **Upload** `kaggle.json` to the Colab files panel (📁 icon in sidebar)
4. **Run All Cells**: The notebook installs dependencies, downloads data via Kaggle API, and executes Q1 → Q2 → Q3 sequentially

## Key Technical Contributions
- **No Data Leakage:** Class weights for CrossEntropyLoss computed strictly on the training set (3,847 samples), never touching validation or test data.
- **Rubric-Exact Split:** 40% test set preserved as mandated. Validation (12%) carved from the 60% training allocation.
- **Class Imbalance Handling:** Weighted CrossEntropyLoss with inverse-frequency weights + F1-Macro as the checkpoint selection metric (not accuracy).
- **Robust Raster Extraction:** `rasterio.windows.Window` with `boundless=True` safely handles boundary patches that extend beyond the raster extent.
- **Full Logging:** `outputs/run_log.txt` captures every pipeline step for post-run analysis.
